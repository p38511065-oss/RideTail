RideTail Fixed Image Name System

Admin image upload/localStorage system disabled for product images.
Now website loads images from ridetail-website/images/ folder.

To change images:
1. Open ridetail-website/images/
2. Replace image with the SAME file name.
3. Upload/deploy website again.
4. Hard refresh browser: Ctrl + F5.
